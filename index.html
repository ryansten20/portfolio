Duck Quest - Game Development Project

Duck Quest is a 2D top-down adventure game inspired by The Legend of Zelda and Cat Quest, created as part of a Software Engineering I course. Built in Unity with C#, the project was a collaborative effort among four teammates, focusing on both gameplay design and software engineering principles. My contributions included developing the menu systems, pause inventory, joystick input for mobile support, HUD elements such as health and currency counters, and integrating sound effects for player actions. The game features puzzles, dungeon exploration, NPC dialogue, and collectible items like keys and chests.

Compiler

For my Compilers course (CS445), I built a full compiler for a simplified programming language called bC. Implemented in C++ using Flex and Bison, the compiler performs lexical analysis, parsing, semantic checks, and code generation. My design included a robust abstract syntax tree, a symbol table for type checking, and logic for handling arrays, functions, and control structures.

DMX-Controlled LED Glasses

For my senior Capstone project, my team and I are designing and developing a pair of LED-integrated glasses that can be controlled through the DMX512 lighting protocol. This project combines embedded systems, digital electronics, and creative design to build a device that bridges traditional entertainment technology with wearable computing. The project centers on an ATmega328p microcontroller, which serves as the core controller for the LED system. Using a DMX512 interface with RS-485 transceivers, the glasses are able to receive industry-standard lighting commands and translate them into dynamic visual patterns across the LED matrix embedded in the lenses. This allows the glasses to integrate seamlessly with existing stage and event lighting systems, meaning they can respond to the same cues as spotlights, strobes, and other show elements.
